# 第2章 網際網路上的攻擊目標為何?

## 1. ipconfig - 這個命令輸出你的當前的網絡配置，包括您計算機的分配IP 地址、默認網關以及有關您的 DNS 服務器的信息。

![image](https://user-images.githubusercontent.com/89304181/221398392-7de81996-1109-49bb-a7f0-4d6497dbaabb.png)

## 2. nslookup - 輸出顯示當前的公共 IP 地址附加到 google.com。

![image](https://user-images.githubusercontent.com/89304181/221398405-4e884b44-29d7-4962-9dae-2b03a2d5d18d.png)

## 3. ping - 該工具向 IP 地址發送一小包信息，然後然後偵聽 ping 的目的地以其自己的數據包響應的信息。

![image](https://user-images.githubusercontent.com/89304181/221398416-5f5a6a8d-b04d-4545-8f99-6bc668e7e83a.png)

## 4. tracert (Trace Route)，此程式可用來追蹤「網際網路通訊協定」(IP) 封包傳遞到目的地所經的路徑。
